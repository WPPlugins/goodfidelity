== Description ==

Complements your website with a administrator of videos and music. It gives many functions to administer and manage video through a bottom bar that does not disturb the theme of your website. Users of your website can create their playlists and save them, can listen to music without interruption, can access millions of music videos. The plugin has a powerful built-in search engine. They can share the videos on Facebook with a single click.

demo: http://www.goodfidelity.com

Paste in post or page:

[good myvideo="IDyoutube"] - example: [good myvideo="k4l3PAKdQCo"]

demo: http://www.goodfidelity.com/una-cancion-legendaria-del-verano-del-67/

[yougood myplaylist="IDPlaylistyoutube"] - example: [yougood myplaylist="PLFgquLnL59alCl_2TQvOiD5Vgm1hCaGSI"]

demo: http://www.goodfidelity.com/electronica-2014/


= Features =

- It has a bar music at the bottom that integrates the theme of your website.
- The videos are searched and are automatically saved in a playlist (when the user is not registered).
- Registered users can create one or more playlists.
- It has a powerful search engine.
- Look for the videos without interrupting the music.
- Each video can be shared on facebook.
- It has a short code to integrate video into your post then be played in the automatic display of the plugin. [good myvideo="IDyoutube"]. Example: [good myvideo="k4l3PAKdQCo"]
- Fits any screen resolution (responsive).
- Fits all languages.
- Share button: facebook, google + and twitter.

== Installation ==

Is easy to install like other plug-ins of Wordpress as you need to just follow the below mentioned steps:

1. Copy el plugin from the downloaded and extracted file.

2. Paste it in wp-Content/plugins folder on your Wordpress Installation 

3. Activate the plugin from Dashboard / Plugins window.

4. Now Plugin is Activated.
